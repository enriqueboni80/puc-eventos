# sistema_autenticacao-api-Nodejs

##### Instalar as dependências
```
npm install
```
##### Rodar a migrate para criar o banco
```
npm run knex migrate:latest
```
##### rodar seed para primeira inserção nas tabelas
```
npm run knex seed:run
```

##### rodar teste Jest
```
npm run secure-mode
```
